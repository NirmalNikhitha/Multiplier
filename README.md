# Multiplier
VLSI CMOS Multiplier implementation using Cadence Virtuoso featuring schematic design and waveform analysis.

## Overview

This repository contains the transistor-level implementation of a CMOS Multiplier designed and simulated using **Cadence Virtuoso**. The project demonstrates custom digital IC design by developing the multiplier schematic and verifying its functionality through transient and differential voltage simulations using the Spectre simulator.


## Features

- CMOS transistor-level multiplier design
- Schematic implementation in Cadence Virtuoso
- Functional verification using transient analysis
- Differential voltage waveform analysis
- Custom VLSI digital circuit design


## Tools Used

- Cadence Virtuoso
- Analog Design Environment (ADE)
- Spectre Simulator

## Simulation Results

### Schematic

The multiplier is implemented using CMOS transistor-level logic in Cadence Virtuoso.

### Transient Waveform

Transient analysis is performed to verify the functional operation of the multiplier by observing the output response for different input combinations.

### Differential Voltage Waveform

The differential voltage waveform is analyzed to observe the voltage difference between output nodes and validate the switching behavior of the circuit.
